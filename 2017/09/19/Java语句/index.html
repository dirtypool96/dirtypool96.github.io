选择

switch语句

  1. default 都不满足时执行
  2. 每个case后都要braek
  3. switch只有 byte short int char 这四种类型

循环

while语句

  • while 后不能加分号

do…while语句

  • 执行完 do 后再进行 while 判断
  • while 后要加分号
  • 至少执行一次

for语句

基本语句

注意：

  1. 操作表达式是 循环后 的操作。
  2. for循环里的变量作用域只在 循环体 中。
  3. 表达式用 “分号” 隔开，多个变量用 “逗号” 隔开。

for嵌套

break 和 continue

  • break 应用范围：选择结构和循环结构
  • continue 应用范围：循环结构

特点：

  1. break和continue语句作用的范围。
  2. break和continue单独存在时，下面可以有任何语句。因为都执行不到。
  3. continue特点：结束本次循环，继续下一次循环。
  4. break特点：直接结束当前循环
